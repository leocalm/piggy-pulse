# PiggyPulse — Categories Page Final Specification

## Purpose

The Categories page is diagnostic. It answers:

- Where is deviation happening?
- At what magnitude?
- What is the trajectory?

No coaching. No prescriptions.

---

## Page Header

Displays:
- Period selector
- Total Budget
- Total Actual
- Total Variance
- % of period elapsed

Time elapsed shown once globally.

---

## Budgeted Categories Section

### Default Sorting
Sort by absolute variance (descending).

### Each Category Row Displays:
- Name
- Budget
- Actual
- Variance (+/-)
- % of budget used
- Progress bar
- Projection (secondary)
- Trend (last 3 periods)

No language like “over” or “under.”
Variance is arithmetic only.

Progress Bar:
- Neutral fill up to 100%
- Extended geometry beyond 100%
- No emotional color coding

Projection text example:
“Projected at current pace: $650”

---

## Unbudgeted Categories Section

Separate section.

Displays per row:
- Name
- Actual spend
- % of total period spend
- Trend (last 3 periods)

No variance.
No projection.
No progress bar.

Sorted by spend descending.

---

## Visual Tone

Balanced density.
Analytical layout (list, not cards).
Clear hierarchy.
No judgmental labels.
